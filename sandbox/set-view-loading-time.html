setViewLoadingTime Demo

This page initializes the Datadog RUM SDK with the set_view_loading_time experimental feature flag enabled. After a simulated 10-second async load, it calls DD_RUM.setViewLoadingTime() to manually report when the view is ready. After the initial loading time is set, you can also test the overwrite scenario by clicking the button that appears. Use the Browser SDK DevTools extension to observe the reported view events.
