# Sandbox

Build the CDN bundles

    yarn build 

Build the app that uses the NPM bundle

    cd test/app
    yarn build 

Run dev server

    yarn dev

Open the playground

- Test SDK CDN cjs: <http://localhost:8080/cdn-cjs.html>
- Test SDK CDN esm: <http://localhost:8080/cdn-esm.html>
- Test SDK NPM: <http://localhost:8080/npm.html>
