Worker workloads

🔢
Sieve of Eratosthenes
Finds all primes up to 50,000. Produces sieve() → inner loop stacks in the profiler.
🌀
Recursive Fibonacci
Computes fib(30) recursively (2× branching). Generates deep, flamegraph-friendly call stacks.
🟦
Matrix multiplication
Multiplies two 80×80 matrices of floats. Tests sustained numeric throughput with tight inner loops.
🌌
Mandelbrot set
Iterates over a 100×100 pixel grid with up to 80 escape iterations per point. Mix of branches and arithmetic.
Requirements: Chromium Canary with flags --enable-features=DocumentPolicyInDedicatedWorker,ProfilerAPIForDedicatedWorker (or --enable-experimental-web-platform-features). The dev server automatically serves both the page and the worker script with the Document-Policy: js-profiling header.
